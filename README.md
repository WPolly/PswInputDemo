# PswInputDemo
简单灵活的密码输入框控件

使用简便，高度可定制化。
## Dependency

Add this in your root `build.gradle` file (**not** your module `build.gradle` file):

```gradle
allprojects {
	repositories {
        maven { url "https://jitpack.io" }
    }
}
```

Then, add the library to your module `build.gradle`
```gradle
dependencies {
    implementation 'com.github.WPolly:PswInputDemo:v1.0'
}
```
